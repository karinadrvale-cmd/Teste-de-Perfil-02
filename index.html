Como isso é calculado

Cada opção corresponde a uma referência temporal dominante:

🔵 Passado: modelo Clássico.
🟢 Presente: modelo Motivador.
🔴 Futuro imediato: modelo Competitivo.
🟡 Futuro estratégico: modelo Estratégico.

Peso: +1.0 ponto para o modelo escolhido.
